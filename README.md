# SwiftPokedex

A production-style Pokédex app built in SwiftUI to demonstrate modern iOS architecture and UX best practices.

---

## 📱 Overview

SwiftPokedex is a personal showcase project built to explore and demonstrate modern iOS development practices using SwiftUI.

The app simulates a real-world production application with a focus on:
- scalable architecture
- clean UI/UX design
- testability

This project is not intended for App Store distribution.

---

## ⚙️ Tech Stack

- SwiftUI
- Swift Concurrency (async/await)
- SwiftData
- Nuke (image loading & caching)
- XCTest
- SwiftLint / SwiftFormat
- GitHub Actions (CI/CD)

---

## 🧠 Architecture

- MVVM pattern
- Feature-based modular structure
- Repository pattern
- Dependency Injection
- Protocol-oriented design

---

## 🚀 Features

### Core
- Pokémon list with pagination
- Pokémon detail screen
- Search & filtering
- Favorites system (SwiftData)

### Performance
- Lazy loading
- Image caching (memory & disk)
- Prefetching strategies
- Pagination optimization

### UX/UI
- Skeleton loading states
- Empty states
- Smooth animations
- Dark mode support
- iPad responsiveness
- Design system

### Accessibility
- Dynamic Type support
- VoiceOver labels
- Reduced Motion support
- Proper contrast handling

### Data
- Local persistence (SwiftData)
- API caching strategy
- Error handling & retry logic

### Testing
- Unit tests (repositories & parsing)
- Light UI snapshot tests
